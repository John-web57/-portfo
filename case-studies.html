01

AI-Powered Cybersecurity System

Real-time threat detection with machine learning and security monitoring.

The Challenge

I wanted to improve the speed and quality of threat detection by reducing noisy alerts and building a system that could identify suspicious activity more intelligently.

My Approach

I designed a pipeline for feature extraction, model training, and security monitoring, then structured the deployment flow so the detection results could be consumed in real time.

Tech Stack

Python Scikit-learn TensorFlow Flask

Outcome

Reduced false positives by 32% and improved detection time by 45%.

02

IoT Water Quality & Flow Control

A smart monitoring and control system for borehole water at Mungaro Estate.

The Challenge

The goal was to monitor water quality continuously and improve flow control with a system that could respond quickly to changing conditions in the field.

My Approach

I designed an IoT workflow that connected sensors, monitoring logic, and real-time alerts so the system could support 24/7 visibility and faster operational decisions.

Tech Stack

Arduino MQTT Node.js MySQL

Outcome

Enabled 24/7 monitoring, automated filter activation, and real-time alerting for key water quality events.

03

Full-Stack Portfolio Backend

A production-ready backend for contacts, storage, and deployment workflows.

The Challenge

I needed a backend that could handle contact submissions reliably, persist messages, and support deployment in a clean and maintainable way.

My Approach

I built an Express-based service with database persistence, email delivery, and deployment-ready structure so the portfolio could move beyond a static site into a full-stack experience.

Tech Stack

Node.js Express Sequelize Render

Outcome

Delivered a secure contact pipeline, persistent storage, and a deployment workflow ready for production updates.
